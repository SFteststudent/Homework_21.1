#Авторизация

***

###Цель: 

**Войти в личный кабинет, чтобы посмотреть список заказов**


###Основной сценарий:


1. Пользователь заходит на сайт и нажимает кнопку  для входа в личный кабинет;
2. Система открывает форму входа с полями для ввода адреса почты и пароля;
3. Пользователь вводит данные и нажимает кнопку для входа;
4. Система проверяет наличие аккаунта с введенными данными и при совпадении открывает страницу личного кабинета клиента.


###Альтернативный сценарий:


3.  (с п.3. основного сценария) Пользователь в форме входа нажимает кнопку для входа по коду из СМС;
4.  Система открывает форму ввода номера телефона и кода;
5.  Пользователь вводит номер телефона и нажимает кнопку для отправки кода;
6.  Система отправляет код на указанный номер телефона;
7.  Пользователь вводит код в поле формы для ввода кода;
8.  Возврат к п.4 основного сценария.


###Исключения:



4. Система проверяет наличие аккаунта с введенными данными и при несовпадении выводит ошибку “аккаунт с такими данными не зарегистрирован”
